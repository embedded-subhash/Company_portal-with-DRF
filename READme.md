# Django HR Management System

A Django-based Employee Database Management System using PostgreSQL.  
This project demonstrates backend development skills including Django ORM, Admin customization, database design, and shell operations.

---

## 🚀 Features

### Department Management
- Create Department
- Update Department
- Delete Department
- Search Departments

### Employee Management
- Create Employee
- Update Employee
- Delete Employee
- Search Employee
- Filter Employee by Department
